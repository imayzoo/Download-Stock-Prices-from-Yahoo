# Download Stock Prices from Yahoo
This script downloads daily historical price data from Yahoo Finances, and makes a csv file.

## Example:
You need to change the next values 
```
nombre = 'AAPL'
inicial = '2017-9-1'
final = '2018-12-31'
```
* inicial = name of stock Apple Inc = APPL
* iinicial = start date
* final = end date
